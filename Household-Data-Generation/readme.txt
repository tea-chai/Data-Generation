input: energy consumption for 62 household in https://github.com/tea-chai/Data-Generation/tree/main/sum

Household_Generation.py takes energy consumption profiles for 62 households

and create new energy consumption dataset for 300 households.

To create a household data ,

It randomly chose a household from https://github.com/tea-chai/Data-Generation/tree/main/sum

diversify each data in this household with random distribution

and store it.
