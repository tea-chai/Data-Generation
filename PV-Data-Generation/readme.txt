
E5 database is used (pV_4KWp_E5.xlsx in https://github.com/tea-chai/Data-Generation/tree/main/PV-Data-Generation/data/4KWp)

to generate new PV energy generation dataset for 150 households,


PV_generation.py randomly picks gneration data in pV_4KWp_E5.xlsx

It expands the data to include 150 households, by applying random distribution to each randomly selected piece of data.
